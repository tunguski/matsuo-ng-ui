Matsuo Angular UI

Angular UI extensions used by Matsuo IT

Github repository

New directives for widgets

Matsuo-ng-ui brings you a bunch of additional directives for composing view.

Simple routing configuration

Automatic view resolution with simple generic rules. Anyone needs five thirty seconds to understand how it works in every single situation.

Form validation automation - with server side errors too!

Angular.js brings fantastic support for form validation. Matsuo-ng-ui extends that to server-side validation and no-form cases too.

Simplified working with modals in 99% of cases.

$dialog service is basing on angular-ui $modal directive and gives you a simple tool for constructing modals in most common situations.

Advanced search query configuration - automated

Searches are the heart of db-centric applications. So when showing db data, elastic search tools are most valuable of all.

Select2 integration

Configure your select 2 in single line with ajax, resource handling etc.

Create bootstrap form fields in one line

With error handling, server-side validation, help information, translation, model rewrite etc.
